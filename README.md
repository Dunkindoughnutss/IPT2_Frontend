# Student Academic Record Management

React project structure for role-based academic record management.

## Folders

- `src/layouts` - role layouts for Student, Faculty, Dean, Chairman, and Registrar
- `src/pages` - login page and role dashboard pages
- `src/modules` - system modules

## Modules

1. User Management
2. Academic Record Management
3. Student Record Management
4. Grade Management
5. Descriptive Analytics
6. Student Kiosks/Portal

## Run

```bash
npm install
npm run dev
```
